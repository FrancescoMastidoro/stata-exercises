# Day 1: Loading Data and First Exploration

**Dataset**: `sysuse auto`
**Estimated time**: ~25 minutes
**Difficulty**: ⬛⬜⬜⬜⬜

## Background
The `auto` dataset contains information on 74 automobile models from 1978, including prices, mileage, weight, and whether the car is foreign or domestic. Learning to load, describe, and summarize a dataset are the very first steps in any quantitative research workflow — you cannot analyze data you have not yet understood.

## Your tasks

1. **Load the dataset** — Use the appropriate command to load the built-in `auto` dataset into memory, then display the first 10 observations so you can see what the raw data looks like.

2. **Inspect the variables** — Run the command that shows you every variable's name, storage type, display format, value label, and variable label all at once. How many variables and observations does the dataset have?

3. **Summarize continuous variables** — Produce summary statistics (mean, standard deviation, min, max) for the variables `price`, `mpg`, and `weight`. Write a one-line comment in your do-file noting which variable has the highest coefficient of variation.

4. **Tabulate a categorical variable** — The variable `foreign` distinguishes domestic from foreign cars. Produce a frequency table for `foreign` that also shows the percentage share and cumulative percentage of each category.

5. *(Bonus)* **Conditional summary** — Without using `if` as a standalone filter command, produce separate summary statistics for `price` and `mpg` for domestic cars only, and then for foreign cars only. Compare the means and note in a comment whether foreign cars in this sample are cheaper or more expensive on average.
