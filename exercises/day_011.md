# Day 11: Visualizing Distributions and Relationships

**Dataset**: `sysuse nlsw88`
**Estimated time**: ~25 minutes

## Background
The `nlsw88` dataset contains 1988 National Longitudinal Survey data on American women, covering wages, occupation, industry, and demographic characteristics. Visualization is a core skill in quantitative labor economics research because it allows you to quickly detect distributional patterns, outliers, and relationships between variables before running any formal analysis.

## Your tasks

1. **Explore the data** — Load the dataset and use `describe` and `summarize` to understand the variables available. Identify three continuous variables and two categorical variables you will use in the tasks below.

2. **Plot a wage distribution** — Draw a histogram of the variable `wage`. Adjust the number of bins to 20 and add a descriptive title. Observe whether the distribution looks symmetric or skewed.

3. **Overlay a normal curve** — Redraw the histogram of `wage`, this time adding a normal density curve on top of it. Use the `normal` option. Does the empirical distribution match the normal approximation?

4. **Scatter plot with grouping** — Create a scatter plot of `wage` (y-axis) against `tenure` (x-axis). Use the `by()` option to produce separate panels for union and non-union workers (`union`). Add axis labels and a title.

5. *(Bonus)* **Bar chart of mean wages by occupation** — Produce a bar chart showing the mean wage for each occupational category (`occupation`). Use `graph bar` with the `over()` option, rotate the x-axis labels so they are readable, and add a note indicating the data source.
